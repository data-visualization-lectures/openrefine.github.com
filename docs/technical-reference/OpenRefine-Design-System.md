[OpenRefine Design System](https://www.figma.com/file/2fGTsMdExFEf4ReUNFzB0V/OpenRefine-Outreachy?type=design&node-id=131%3A315&mode=design&t=sqE5Yl0QJtw5TAHy-1) は、ツール内のインタラクティブ要素を丁寧に分析し、コンポーネントのスタイル・用途・ユーザーフロー別の統合方法を示すリファレンスです。一貫性と使いやすさを保つための規範的ドキュメントでもあります。Figma ファイルのページナビから、カラー、タイポグラフィ、ボタン、ダイアログ、インタラクションシーケンスなどのガイドラインを参照してください。

## アクセス方法

**編集用に複製**: デザインシステムは誰でも閲覧できます。編集したい場合は自分の Drafts に複製してください。そこから独自 Figma ファイルへコンポーネントをコピーすることも可能です。

**編集権限の申請**: 活発な貢献者には直接編集権限を付与できる場合があります。OpenRefine フォーラムの開発・デザインカテゴリで気軽に問い合わせてください。
