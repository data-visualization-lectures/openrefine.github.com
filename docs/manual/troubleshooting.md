## よくある質問 {#frequently-asked-questions}

よくある質問と回答は GitHub の [FAQ ページ](https://github.com/OpenRefine/OpenRefine/wiki/FAQ) にまとめて公開しています。

そこで問題や解決策が見つからない場合は、以下のコミュニティリソースで過去の議論を探したり、新しく質問してみてください。  

## コミュニティ {#community}

### 問題が起きたとき {#if-youre-having-a-problem}
*   Web で既知の問題か検索してください。[フォーラム](https://forum.openrefine.org) や旧メーリングリスト（[openrefine](https://groups.google.com/forum/#!forum/openrefine)、[openrefine-dev](https://groups.google.com/forum/#!forum/openrefine-dev)）、あるいは [Stack Overflow](https://stackoverflow.com/questions/tagged/openrefine) に情報があるかもしれません。
*   不具合を報告する:
    *   [フォーラム](https://forum.openrefine.org) で新しいスレッド（会話）を立てる。
    *   もしくは直接 [GitHub Issue を作成](https://github.com/OpenRefine/OpenRefine/issues/new/choose) する。

### 貢献したいとき {#if-you-want-to-contribute}
*   Weblate を使って [UI の翻訳](../technical-reference/translating-ui) に協力する。
*   ドキュメント技術編の [コントリビューションガイド](../technical-reference/contributing) を読む。
*   機能要望を [フォーラム](https://forum.openrefine.org) や [GitHub Issues](https://github.com/OpenRefine/OpenRefine/issues/new/choose) で共有する。
*   [ブログ記事やガイド、チュートリアル](https://github.com/OpenRefine/OpenRefine/wiki/External-Resources) をリストに追加する。
*   隔年で実施しているユーザーサーベイに回答する。
