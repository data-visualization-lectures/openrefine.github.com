# プロジェクトの資金概要

OpenRefine はボランティアの貢献に加え、複数の資金源によって開発を継続しています。多様な資金により、コアチームの維持・基盤インフラ・コミュニティへの働きかけを支えています。

## 運営予算

OpenRefine のコアチーム（インフラ保守、コミュニティ対応、ボランティアレビュー、セキュリティ報告の対応、拡張やリコンシリエーション生態系の管理を担うパートタイムの少人数チーム）を支えるため、皆さんの支援を必要としています。

年間の運営予算は **14 万ドル**（月あたり約 1.15 万ドル）で、内訳は以下のとおりです。
* 月 7,500 ドル: 開発・コントリビューター支援リード
* 月 2,500 ドル: プロジェクト調整
* 月 70 ドル: 必須ソフトウェアライセンス（Figma、Discourse）
* 寄付額の 15%: 財務ホストである Code for Science and Society への事務・コンプライアンス費

追加の資金があれば、コミュニティイベントの開催、アウトリーチ、特別プロジェクトにも取り組めます。OpenRefine ではコミュニティの関心や優先度に基づく [goalpost のリスト](/docs/technical-reference/development-roadmap) を公開しており、達成のためのリソースを探しています。組織とのパートナーシップや寄付でご支援ください。

## 個人・法人からの寄付

OpenRefine は米国の慈善団体 [Code for Science & Society](https://www.codeforsociety.org/)（501(c)(3)）の財務スポンサーシップのもとで運営されており、米国法の範囲で寄付控除の対象となります（Tax ID: 81-3791683）。

**寄付方法**

- [GitHub Sponsors](https://github.com/sponsors/OpenRefine)
- [寄付フォーム](/donate)（クレジットカード・PayPal・銀行振込、上限 25,000 ドル）
- より高額の場合は Code for Science & Society の [寄付ページ](https://www.codeforsociety.org/donate)

現在の [支援者一覧](/backers) に感謝しています。
